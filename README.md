 # AI---Driven-Sales-Forecasting-Dashboard
📊 Power BI Project: AI-Driven Sales Intelligence Dashboard

I recently built this interactive dashboard to analyze sales performance and uncover hidden patterns using Power BI’s AI capabilities.

🔹 KPI monitoring for Sales, Profit, Orders, and Profit Margin
🔹 Region-wise and Category-wise performance analysis
🔹 Dynamic filters for deeper business insights

Tech Stack:
Power BI | DAX | Data Modeling | AI Visuals

This project helped me explore how data visualization and AI features can work together to support smarter business decisions.
